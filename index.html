How it works

  • Use the toolbar above the formula to insert items with one click — operators, variables, functions, and literals all go in at the current cursor.
  • The blue, pulsing + button is the cursor — your next insert lands there. Click any other + (or empty arg slot) to move the cursor.
  • Picking a function or ( ) auto-drills the cursor into its first arg so you can keep typing left-to-right.
  • Click any pill to edit or replace it. Hover a function/group to reveal an × that deletes it whole.
  • Commas are structural — they appear automatically between function args, you never type them.
  • Saved JSON updates live; variables emit as "@id", literals stay as raw strings.
